# QOL-Crafting-Recipes
This datapack adds and changes tons of recipes that just make sense.

-Proper stair math. All stair recipes output 8 stairs.
-Uncrafting of slabs/stairs. All stairs and slabs can be crafted back into the original resource.
-Flint stripping. Can get stripped wood using flint.
-Improved dyeing. dyeing wool, concrete powder, terracotta and glass are all improved.
-Shortcut recipes for several things, IE: Chests, sticks, etc. 


If you find anything we need to change/add let us know in the issues tracker.

In the future, may be adding peaceful mob recipes.
